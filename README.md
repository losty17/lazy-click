# Command-Task

Command-Task is a high-performance Go TUI for project management systems, starting with ClickUp.

## Current Status

Scaffolded foundation:

- Provider abstraction contracts.
- Unified domain models.
- Bubble Tea app shell.
- SQLite cache wiring scaffold.
- Living roadmap in `ROADMAP.md`.

## Run

```bash
go run ./cmd/command-task
```
